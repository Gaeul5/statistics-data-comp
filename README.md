# 국내 관광 소멸 위기 지역 분석 및 정책 제안

## 프로젝트 요약

* **목적**: 관광객 소멸 지역 선정을 통한 지방 소멸 위기 대응 정책 제안 


* **데이터**: 방문자 수, 검색량, 이동량, SNS 언급량 등 (2023년 기준) 


* **주요 기법**: K-Means 클러스터링(지역 분류), XGBoost(방문자 수 예측) 



## 분석 결과

### 1. 최적 모델

* **XGBoost**: Grid Search 최적화 결과 **정확도 96.11%** 달성 



### 2. 소멸 위기 지역 TOP 5 (2025년 예측)

방문자 감소율이 가장 높게 예측된 지역들입니다.

1. **전남 담양군** (230.80%) 


2. **경북 울릉군** (210.43%) 


3. **대구 군위군** (103.01%) 


4. **전남 장성군** (92.07%) 


5. **충남 공주시** (68.78%) 



## 정책 제안

* 소멸 위기 지역 공표를 통한 지자체 위기의식 환기 


* 지역 특화 관광 상품 개발 및 맞춤형 예산 지원 요청
